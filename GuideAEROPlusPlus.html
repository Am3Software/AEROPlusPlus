overview
AERO++
C++ header-only API for aircraft preliminary design — aerodynamics, weight & balance, inertia, performance, statistical analysis, 3D visualization, and data export.

Features

FeatureDescription
Header-only libraryEasy integration — just include the headers.
EigenLinear algebra (included).
Gnuplot integration2D data visualization via gnuplot-iostream.
VTK-based 3D visualizationInteractive rendering and multi-view PNG export (AircraftPlotter).
Per-component color mappingAssign colours to each aircraft component by name.
OpenVSP DegenGeom CSV parserParse OpenVSP degenerate geometry output.
OpenCASCADE integrationSTEP/IGES geometry import.
Cross-platformLinux, macOS, Windows (MSYS2).

Installation
Platform-specific instructions for installing all required and optional dependencies.

Linux (Ubuntu / Debian)

sudo apt-get update
sudo apt-get install build-essential cmake libboost-all-dev gnuplot libvtk9-dev

macOS

brew install cmake boost gnuplot vtk

Windows (MSYS2 — Recommended)

1. Download and install MSYS2 from msys2.org.

2. Open the MSYS2 MinGW 64-bit terminal and run:

pacman -Syu                                    # Update package database
pacman -S mingw-w64-x86_64-gcc
pacman -S mingw-w64-x86_64-cmake
pacman -S mingw-w64-x86_64-boost
pacman -S mingw-w64-x86_64-gnuplot
pacman -S mingw-w64-x86_64-vtk                 # 3D visualization
pacman -S mingw-w64-x86_64-opencascade         # STEP/IGES import (optional)
pacman -S mingw-w64-x86_64-python              # Python bindings (optional)
Always use the MSYS2 MinGW 64-bit terminal, not the default MSYS2 terminal.

Quick Start
Clone the repository, build with CMake, and run the test suite in minutes.

Build with CMake

git clone https://github.com/Am3Software/AEROPlusPlus.git
cd AEROPlusPlus
mkdir build && cd build
cmake ..
cmake --build .

CMake will automatically download OpenXLSX and tinyxml2 if not found, auto-detect VTK, and print a configuration summary. Optional features that are not available are skipped with a warning (no hard failures).

Run the tests

# Linux / macOS / MSYS2
./TestVSPCreator
./PowerTestPropellerAircraft

# Windows CMD
TestVSPCreator.exe
PowerTestPropellerAircraft.exe

Use in your project

Method 1 — Include directly

#include "AEROPlusPlus/include/Plotter3D.h"
#include "AEROPlusPlus/include/DegenGeomParser.h"
#include <Eigen/Dense>

Method 2 — CMake integration

include_directories(/path/to/AEROPlusPlus/include)

find_package(VTK REQUIRED)
find_package(Boost COMPONENTS iostreams system filesystem REQUIRED)

add_executable(your_program your_code.cpp)
target_link_libraries(your_program ${Boost_LIBRARIES} ${VTK_LIBRARIES})
vtk_module_autoinit(TARGETS your_program MODULES ${VTK_LIBRARIES})

Custom VTK path

If VTK is installed in a non-standard location:

cmake -DVTK_DIR=/path/to/vtk/cmake ..
